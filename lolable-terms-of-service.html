2. Description of Service

Lolable is a non-commercial Discord bot built for League of Legends community servers. It allows Discord users to voluntarily link their Riot ID to their Discord account and access in-server statistics, rankings, and match information.

Features include:

  • /bagla – Link a Riot ID and tag to your Discord account
  • /profil / /me – View your profile, rank, champion mastery, and recent match stats
  • /sonmac – Show a summary of your last ranked match
  • /live – View live game scouting when Riot Spectator data is available
  • /leaderboard – Rank linked server members by SoloQ LP and rank
  • /duo – Find similarly ranked linked users in the server
  • /compare – Compare two linked users side by side
  • /gunluk – Daily ranked performance summary
  • /championpool – Champion pool and main pick analysis
  • /feed – Optional match, rank, and streak notifications for a designated channel
  • /help / /panel – Help information and interactive command panel
