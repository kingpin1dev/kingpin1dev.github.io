Lolable

Privacy Policy

Last updated: June 17, 2026

1. Overview

Lolable is a non-commercial Discord bot that helps League of Legends communities view in-server stats, rankings, and match information. This Privacy Policy explains what data we collect, why we collect it, and how it is used and protected.

We collect only what is necessary to provide the bot's features. We do not sell, trade, or share your data with third parties beyond what is strictly required to operate the service.

2. Data We Collect

When you use Lolable, the following information may be stored:

  • Discord user ID – to identify your account across commands
  • Discord server (guild) ID – to scope data to the correct server
  • Discord channel ID – only if you or your server admin configures a /feed notification channel
  • Riot gameName and tagLine – the Riot ID you link via /bagla
  • Riot PUUID – a unique identifier provided by the Riot Games API to look up match data
  • Encrypted summoner ID – when available from the Riot API
  • Platform and region settings – your selected server region (e.g. EUW, TR)
  • Profile icon ID and summoner level – for display in profile commands
  • Ranked snapshot data – tier, division, LP, wins, losses, and recent win rate, used for leaderboard and comparison features
  • Last seen match IDs – used only to detect new matches for feed/notification purposes
  • Minimal server configuration – settings needed for features such as feed channels

3. Data We Do NOT Collect

Discord

  • Message content
  • Direct or private messages
  • Username or display name
  • Email address

Personal / Other

  • Real names
  • Passwords
  • Payment or financial information
  • Raw Riot API responses for resale or public dumping

4. How We Use Your Data

Collected data is used exclusively to provide Lolable's features:

  • Displaying your profile, rank, and match statistics via /profil, /me, /sonmac, /live
  • Generating server leaderboards via /leaderboard
  • Enabling duo matching and player comparisons via /duo, /compare
  • Showing daily performance summaries via /gunluk
  • Analyzing champion pool data via /championpool
  • Sending match, rank, and streak notifications if a feed channel is configured via /feed

Your data is never used for advertising, profiling, gambling, betting, boosting, cheating services, or any form of competitive integrity abuse.

5. Riot Games API Data

Game data displayed by Lolable is retrieved in real time from the Riot Games API and Data Dragon. This data is used only to power the features described above and is not stored beyond what is listed in Section 2.

Lolable is not affiliated with, endorsed by, sponsored by, or approved by Riot Games. League of Legends and Riot Games are trademarks or service marks of Riot Games, Inc. Use of the Riot Games API is subject to Riot's Developer Policies.

6. Data Sharing

We do not sell, rent, or trade your data. Data may only be shared in the following limited circumstances:

  • Infrastructure providers – hosting and database services necessary to run the bot, bound by their own privacy policies
  • Legal requirements – if required by law, court order, or governmental authority

No other third-party sharing occurs.

7. Data Retention and Deletion

Data is retained for as long as:

  • The bot remains active in your Discord server, or
  • Your Riot ID remains linked via /bagla

You can request deletion of your data at any time. Upon a valid request, your linked account data (Riot ID, PUUID, ranked snapshots, match IDs) will be removed. Server configuration data will be removed when the bot is kicked from the server or upon an admin request.

To request data deletion: Contact us via Discord (see Section 11) and provide your Discord user ID. We will process your request as soon as reasonably possible.

8. Security

We take reasonable precautions to protect stored data:

  • API keys and tokens are never exposed publicly
  • Data is stored in a private database with restricted access
  • Summoner IDs are stored in encrypted form where applicable

No system can be guaranteed 100% secure. In the unlikely event of a data breach affecting your information, we will make reasonable efforts to notify affected users.

9. Your Rights

You have the right to:

  • Access – request a summary of the data stored about you
  • Correction – update or relink your Riot ID at any time using /bagla
  • Deletion – request that your data be erased (see Section 7)
  • Opt out – simply do not use /bagla; no data is stored about you unless you voluntarily link your account

To exercise any of these rights, contact us through the channels listed in Section 11.

10. Children and Minors

Lolable is intended for Discord users who meet Discord's minimum age requirements (13 years of age or older, or older where required by local law). We do not knowingly collect data from users who do not meet these requirements. If you believe a minor has linked an account without appropriate consent, please contact us so we can remove their data.

11. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page will reflect the most recent revision. Continued use of Lolable after changes are posted constitutes acceptance of the updated policy.
